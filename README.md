Runnable setup for [IntelliHire (3).jsx](IntelliHire%20(3).jsx). Live: [delta-virus.vercel.app](https://delta-virus.vercel.app)

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
